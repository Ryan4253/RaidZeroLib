rz::AdaptivePurePursuitController class
#include <include/RaidZeroLib/api/Chassis/AdaptivePurePursuitController.hpp>

The AdaptivePurePurePursuitController is a controller that drives the robot along a path using the pure pursuit algorithn.

Contents

  • Reference
    • Public types
    • Constructors, destructors, conversion operators
    • Public functions

In each iteration, the controller computes the lookahead point on the path that is one lookahead distance away from the robot. The robot then drives towards the point at constant curvature given the target velocity. This process is repeated until the end of the path is reached.

Parameters
